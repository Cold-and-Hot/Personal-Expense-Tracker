# 💰 Personal Expense Tracker (ระบบบันทึกรายจ่ายส่วนตัว)

แอปพลิเคชันเว็บแบบ Full-stack สำหรับบันทึกและจัดการรายจ่ายส่วนตัว ช่วยให้คุณติดตามสถานะทางการเงินได้อย่างง่ายดาย

## ✨ ฟีเจอร์หลัก (Features)

- ✅ บันทึกรายการรายจ่าย (ชื่อรายการ, จำนวนเงิน, หมวดหมู่)
- 📊 สรุปยอดรวมค่าใช้จ่ายทั้งหมดแบบ Real-time
- 🗑️ ระบบลบรายการพร้อม Custom Confirm Modal (Pop-up)
- 📱 รองรับการแสดงผลแบบ Responsive (สวยงามทั้งคอมและมือถือ)
- 🔐 ระบบความปลอดภัย ซ่อนข้อมูล Database ด้วย `.env`

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL
- **Environment Management:** Dotenv
